## Developing
To build the scripts:
```bash
npm install
bundle install
npm run build:script
npm run build:style
```

To run the docs locally:
```bash
# installation
cd docs
npm install

# running
npm run build:script
open index.html
```

In case you'd like to watch for changes:
```bash
# on ./doc/ root
npm run watch:script
npm run watch:style
```
